jQuery 의 연결

해당 프로젝트에 다운로드 받아 연결하거나 CDN 방식으로 연결한다.

항상 최신판을 유지할 경우는 https://code.jquery.com/jquery-latest.js 로 연결

다만, 현재 진행하거나 예전에 진행했던 프로젝트일 때 유지보수상 latest.js는 사용하지 않는 것이 좋다.



jQuery의 적용

해당 요소의 아래에 전개할 경우는 다음과 같다.

                $(function(){    
                    $("#con").text("김기태");    
                });


해당 요소의 위에 전개할 경우는 다음과 같다.

                $(doucment).ready(function(){
                    $("#con").text("김태엽"); 
                });
